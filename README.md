# Dive Evolution (Next.js Migration)

Migracion de Angular a Next.js conservando estructura, rutas, estilos y consumo de APIs.

## Ejecutar

```bash
npm install
npm run dev
```

Abrir en `http://localhost:3000`.

## Rutas

- `/`
- `/nosotros`
- `/san-cristobal`
- `/santa-cruz`
- `/isabela`
- `/cursos`
- `/cursos/[id]`
- `/tours/[id]`
- `/contacto`
